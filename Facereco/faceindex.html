Domain: AI/ML Project

Our project is based on Face Recognition concept.It will be computer application capable of detecting, identifying or verifying human faces from an image or video captured using a digital camera.

Why I choose this project?

To start our AI/ML journey, we were looking forward to work on projects, real time screnarios. Then while learning about different libraries in Python. We found it very fascinating that we can detected faces with python libraries. We took it as a challenge to create a database to recognize the face of the people. Several Libraries have been used in this project. We can utilise this application to monitor attendance of employees/staff at various organisations. We can also use this application as security measure in case of impersonate.
